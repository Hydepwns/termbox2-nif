## Getting Started


### Prerequisites

You will need to have a
* modern erlang  (Version 22+)
* make 
* c compiler.

### Installation

TBD: Should we get this from hex.pm show how to use it as a dep ?


Manually:

1. Clone this repo.
   ```sh
   git clone https://github.com/wmealing/termbox2-nif.git
   ```
2. Build the library 
   ```sh
	make
   ```
